# Dorisoy.MotorSimulator

该项目对 Bien-Air 四款无刷无感电动马达（MX-i LED PLUS、MX-i LED、MX2、MCX LED）的特性曲线进行了可视化，覆盖种植、外科、修复、预防及根管治疗等牙科应用场景。系统支持动态配置减速比，直观呈现马达性能。其核心意义在于：通过设定目标性能指标，辅助开发人员精准设计符合临床需求的马达驱动方案，确保输出特性与应用要求一致。

## 六、驱动方案设计要求

### 6.1 驱动拓扑

采用 **三相全桥逆变器** 拓扑结构（集成于驱动IC内部）：

```
         +24V
          │
    ┌─────┼─────┐
    │     │     │
   Q1    Q3    Q5     ← 集成于DRV8316
    │     │     │
    ├──U──┼──V──┼──W──→ 牙科手机电机
    │     │     │
   Q2    Q4    Q6     ← 集成于DRV8316
    │     │     │
    └─────┼─────┘
          │
         GND
```

### 6.2 无感驱动算法要求

| 控制模式 | 转速范围 | 算法要求 | 备注 |
|----------|----------|----------|------|
| **启动模式** | 0～启动 | I-F启动（开环电流-频率控制） | 带负载可靠启动 |
| **低速FOC** | 100～2,000 RPM | FOC + 滑模观测器（SMO） | 低速稳定、高扭矩 |
| **高速FOC** | >2,000 RPM | FOC + BEMF观测器 | 高速高效 |

**关键算法要求：**

1. **启动算法（种植手术关键）**
   - 必须支持带负载启动（≥30 N·cm负载）
   - 启动时间 ≤0.5s（从静止到100 RPM）
   - 启动失败自动重试（最多3次）
   - 启动电流限制≤峰值电流的80%

2. **低速稳定性（种植手术关键）**
   - 100 RPM 稳定运行，转速波动≤±3%
   - 采用 FOC + 滑模观测器（SMO）方案
   - 支持低速大扭矩（100RPM@30 N·cm）

3. **扭矩控制（种植手术关键）**
   - 扭矩限制响应时间 ≤10ms
   - 扭矩测量精度 ≤±5%
   - 达到扭矩限制平滑降速，无冲击

### 7.4 保护电路要求（医疗级加强）

| 保护类型 | 保护阈值 | 动作 | 恢复方式 |
|----------|----------|------|----------|
| **过流保护（OCP）** | >6A（峰值，<1ms） | 立即关断 | 自动恢复（3次后锁定） |
| **软件过流** | >3A（持续>100ms） | 降功率→关断 | 自动恢复 |
| **过压保护（OVP）** | >30V | 关断 | 自动恢复 |
| **欠压保护（UVP）** | <20V | 关断 | 自动恢复 |
| **过温保护（OTP）** | >100℃（芯片）/ >80℃（板级） | 降功率→关断 | 冷却后恢复 |
| **堵转保护** | 堵转时间>0.3s | 自动关断 | 手动复位 |
| **漏电流监测** | 漏电流>1mA | 关断+报警 | 手动复位 |
| **看门狗** | 软件异常 | 系统复位 | 自动重启 |
